HR Analytics & Employee Attrition Dashboard
Project Overview

This Power BI dashboard analyzes employee attrition trends and workforce insights using HR data. The dashboard helps identify factors affecting employee turnover and supports HR decision-making.

Key Insights
Research & Development department shows highest attrition.
Employees working overtime are more likely to leave.
Younger employees show higher attrition trends.
Salary and job satisfaction significantly impact attrition.
